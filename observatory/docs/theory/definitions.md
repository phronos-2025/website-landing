# Key Definitions

This document provides precise definitions for core concepts used throughout P.H.R.O.N.O.S. research.

## Strategic Foresight

**Strategic Foresight** is the systematic anticipation of future possibilities and their implications for present decision-making, extending beyond 3-year planning horizons.

## Signal-to-Noise Ratio

The **Signal-to-Noise Ratio** represents the proportion of actionable, high-quality information to irrelevant or misleading data in an information ecosystem.
